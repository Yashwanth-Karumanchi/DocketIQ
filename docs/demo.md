# DocketIQ Demo Script

Use this order for a clean evaluator demo.

## 1. Login

Log in with Google.

Show:

- Allowed Google account
- Dashboard loads
- User profile visible

---

## 2. Dashboard

Show:

- Active Cases
- High Priority
- Open Tasks
- Pending Actions
- Documents

Click:

- High Priority
- Open Tasks
- Documents

Search:

```txt
Henry
```

Open a case.

---

## 3. Case Workspace

Show:

* Case overview
* Missing items
* Documents
* Timeline
* Agent reports
* Communication Autopilot
* Case graph
* Chat button

Ask chat:

```txt
Summarize this case.
```

---

## 4. Agents

Run:

```txt
Missing Items
Readiness
Next Best Action
Relationships
```

Show:

* Agent result
* Updated tasks
* Report history

---

## 5. Document Upload

Upload a sample PDF.

Ask chat:

```txt
What changed after the upload?
```

---

## 6. Communication Autopilot

Click Refresh Suggestions.

Expand a suggestion.

Click Convert to Pending Email.

Show:

* Pending action created
* Nothing sent yet

---

## 7. Gmail Action

Ask chat:

```txt
Send the client an email asking for the police report and treatment records.
```

Show:

* Pending action
* Confirm / Cancel buttons

Click Confirm only when ready.

---

## 8. Calendar Action

Ask chat:

```txt
Schedule a 30-minute consultation with the client tomorrow at 3 PM Mountain Time.
```

Show:

* Pending calendar action
* Confirm button

Click Confirm.

Open Calendar page.

Show event.

---

## 9. Reports

Go to Reports.

Search:

```txt
handoff
```

Show:

* Agent runs
* Handoff records

---

## 10. New Case

Go to New Case.

Create a case with intake form.

Show:

* Case opens automatically
* Timeline event created
* Missing tasks created
* Chat can summarize it

---

## Best Closing Line

DocketIQ shows a full agentic legal-operations workflow: secure login, case intake, document intelligence, agent workflows, human-approved Gmail and Calendar actions, and audit-ready reports.
